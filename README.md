# Interior Website

A modern and responsive Interior Design Website built using HTML and CSS. The project showcases elegant interior design services, featured projects, image galleries, and a visually appealing user interface.

## 🚀 Live Demo

https://hoddurshardha-code.github.io/interior_website/

## 🛠️ Technologies Used

* HTML5
* CSS3

## ✨ Features

* Responsive design
* Modern and clean user interface
* Image gallery section
* Interior design showcase
* Smooth layout and navigation
* Cross-browser compatible

## 📂 Project Structure

```text
interior_website/
├── index.html
├── style.css
└── images/
```

## 🎯 Purpose

This project was created to practice frontend web development skills and design attractive, user-friendly website layouts using HTML and CSS.
